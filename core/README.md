## ✍️ Usage

### Frontend

```sh
cd client
```

#### Dev

```sh
# Install dependencies
pnpm i

# Development
pnpm run dev
```

#### UI Kit

```sh
pnpm run storybook
```

### Backend

#### Dev

```sh
cd server
```

```sh
# Install dependencies
pnpm i

# Development
pnpm run start
```
