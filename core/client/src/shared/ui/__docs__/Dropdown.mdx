# UiDropdown

The **UiDropdown** component provides a versatile dropdown menu that supports various directional placements, along with customizable slots for header, trigger, and content.

<Canvas of={DropdownStories.DefaultDropdown} />

### Props

<Markdown>
  {`
| Prop      | Description                                                                                             |
|-----------|---------------------------------------------------------------------------------------------------------|
| direction | Defines the dropdown's alignment. Options are: __left__, __center__ (default), and __right__.           |
`}
</Markdown>

<Canvas of={DropdownStories.DropdownExampleProps} />

### Slots

<Markdown>
  {`
| Slot     | Description                                                                                             |
|----------|---------------------------------------------------------------------------------------------------------|
| trigger  | Slot for the trigger element that opens or closes the dropdown.                                         |
| header   | Optional slot for the header section within the dropdown content, ideal for titles or icons.            |
| content  | Slot for the main dropdown content. This area closes the dropdown when clicked.                         |
`}
</Markdown>
