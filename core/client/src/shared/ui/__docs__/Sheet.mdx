# UiSheet

The **UiSheet** component is a versatile sheet panel that can be used for side-drawer-like interactions. It provides various customization options such as size, overlay, and transitions.

<Canvas of={SheetStories.DefaultSheet} />

### Props

<Markdown>
  {`
| Prop              |Description                                                                                 |
|-------------------|--------------------------------------------------------------------------------------------|
| overlay           | Enables or disables the overlay background.                                                |
| maxWidth          | Sets the maximum width of the sheet.                                                       |
| transitionDuration| Controls the duration of the transition effect in seconds.                                 |
| minWidth          | Sets the minimum width of the sheet.                                                       |
| overlayClickClose | Determines if clicking on the overlay closes the sheet.                                    |
  `}
</Markdown>

### Slots

<Markdown>
  {`
| Slot      | Description                                                                 |
|-----------|-----------------------------------------------------------------------------|
