# UiTextarea

The **UiInput** component is a customizable textarea that can be easily integrated with `v-model` for value binding.

<Canvas of={TextareaStories.DefaultTextarea} />

### Props

<Markdown>
  {`
| Prop        | Description                                                                                                     |
|-------------|-----------------------------------------------------------------------------------------------------------------|
| modelValue  | The textarea field value. **Use __v-model__ to bind this prop to a data property.**                                |
| defaultValue| The default value                                                                                               |
| class       | Use for customize input with unocss by type of HTMLAttributes['class']                                          |
  `}
</Markdown>
